# todo_riverpod

## はじめに

Firebase、Riverpod の学習を目的に作った ToDo アプリです。
Firestore にアクセスしデータを保存、削除、読み込みなどが出来るようにしています。

## 使用技術、バージョン

- Flutter 3.0.4
- flutter_riverpod: ^1.0.3
- firebase_core: ^1.19.2
- cloud_firestore: ^3.4.0
- uuid: ^3.0.6
